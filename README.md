# 🚀 Job Tracker (Full Stack Application)

A full-stack Job / Internship Tracker web application to manage job applications efficiently.

🌐 **Live Demo:** https://job-tracker-hazel-ten.vercel.app/
🔗 **Backend API:** https://job-tracker-backend-on0u.onrender.com

---

## ✨ Features

* 🔐 User Authentication (Signup / Login with JWT)
* 📊 Dashboard with job statistics
* 💼 Add / Edit / Delete job applications
* 📍 Track company, role, location & status
* 🔎 Clean and responsive UI
* 🌐 Fully deployed (Frontend + Backend)

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS
* Axios

### Backend:

* Node.js
* Express.js
* MongoDB (Atlas)
* JWT Authentication

### Deployment:

* Frontend → Vercel
* Backend → Render

---

## 📂 Project Structure

job-tracker/
│
├── frontend/    # React frontend
├── server/    # Node.js backend

---

## ⚙️ Setup Instructions (Local)

### 1. Clone repo

git clone https://github.com/your-username/job-tracker.git
cd job-tracker

---

### 2. Setup Backend

cd server
npm install

Create `.env` file:

MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key

Run server:

npm start

---

### 3. Setup Frontend

cd frontend
npm install
npm start
